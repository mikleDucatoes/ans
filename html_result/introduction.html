Введение¶

«Умный» графин iWine обладает встроенным Web-сервером, и имеет REST API для взаимодействия с устройствами внутри локальной сети. Используя функции из API, возможно, разрабатывать собственные мобильные приложения, следуя инструкциям и правилам обмена данными по протоколам HTTP и REST API. Данный функционал протестирован и предназначен изготовителем для разработчиков мобильных приложений, которые управляют состоянием iWine, используя набор API – функций. Взаимодействие происходит поверх протокола HTTP, так как HTTP достиг высокого распространения, потому REST API, организованный поверх HTTP, возможно использовать практически из любого языка программирования. Все вызовы функций API – интерфейса представляют собой HTTP – запросы, методов GET или POST ниже будет подробнее представлено краткое описание использования API (подробнее о протоколе, содержании заголовков http-пакетов и т.п. вам следует уточнить в спецификации протокола HTTP).
